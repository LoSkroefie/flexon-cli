FLEXON CLI Tutorials

This tutorial teaches you how to implement FLEXON CLI in your projects using various programming languages. Each section explains how to use CLI commands for encoding, decoding, inspecting, and validating data in FLEXON format.

1. Understanding FLEXON CLI Commands

Start by familiarizing yourself with the core FLEXON CLI commands:

  • Encoding: Convert JSON to FLEXON binary format. 
    flexon-cli encode input.json output.flexon
  • Decoding: Convert FLEXON binary back to JSON. 
    flexon-cli decode input.flexon output.json
  • Inspecting: View the structure and content of a FLEXON file. 
    flexon-cli inspect input.flexon
  • Validating: Check if a FLEXON file matches a schema. 
    flexon-cli validate input.flexon schema.json

2. Python Integration

Follow these steps to integrate FLEXON CLI in Python:

  1. Install Python from python.org.
  2. Create a script named flexon_example.py with the following content:
    3. 
import subprocess

# Encode JSON to FLEXON
subprocess.run(["flexon-cli", "encode", "input.json", "output.flexon"])

# Decode FLEXON to JSON
subprocess.run(["flexon-cli", "decode", "output.flexon", "decoded.json"])
  3. Run the script: 
    python flexon_example.py

3. VB.NET Integration

  1. Open Visual Studio and create a new VB.NET Console Application.
  2. Replace the default code with: 
    
Imports System.Diagnostics

Module Module1
    Sub Main()
        Dim command As String = "flexon-cli"
        Dim arguments As String = "encode input.json output.flexon"

        Dim startInfo As New ProcessStartInfo() With {
            .FileName = command,
            .Arguments = arguments,
            .RedirectStandardOutput = True,
            .UseShellExecute = False,
            .CreateNoWindow = True
        }

        Using process As Process = Process.Start(startInfo)
            Dim result As String = process.StandardOutput.ReadToEnd()
            Console.WriteLine(result)
        End Using
    End Sub
End Module
  3. Run the application to test the encoding process.

4. Java Integration

Use the following steps for Java:

  1. Create a file named FlexonExample.java with the following content:
    2. 
import java.io.BufferedReader;
import java.io.InputStreamReader;

public class FlexonExample {
    public static void main(String[] args) {
        try {
            String command = "flexon-cli encode input.json output.flexon";
            Process process = Runtime.getRuntime().exec(command);

            BufferedReader reader = new BufferedReader(new InputStreamReader(process.getInputStream()));
            String line;
            while ((line = reader.readLine()) != null) {
                System.out.println(line);
            }
            process.waitFor();
        } catch (Exception e) {
            e.printStackTrace();
        }
    }
}
  2. Compile and run the program: 
    javac FlexonExample.java && java FlexonExample

5. Node.js Integration

To use FLEXON CLI in a Node.js project:

  1. Create a file named flexon_example.js and add this code:
    2. 
const { exec } = require('child_process');

// Encode JSON to FLEXON
exec('flexon-cli encode input.json output.flexon', (error, stdout, stderr) => {
    if (error) {
        console.error(`Error: ${error.message}`);
        return;
    }
    console.log(`Output: ${stdout}`);
});
  2. Run the script using: 
    node flexon_example.js

PHP Tutorial

Integrate FLEXON CLI in a PHP application by executing system commands:

  1. Ensure FLEXON CLI is installed and available in your system's PATH.
  2. Create a PHP script (e.g., flexon_example.php) and add the following code:
    3. <?php
// Encode JSON to FLEXON
$output = shell_exec('flexon-cli encode input.json output.flexon');
echo "Encode Output: " . $output . "\n";

// Decode FLEXON to JSON
$output = shell_exec('flexon-cli decode output.flexon decoded.json');
echo "Decode Output: " . $output . "\n";
?>
  3. Run the script:
    4. php flexon_example.php
